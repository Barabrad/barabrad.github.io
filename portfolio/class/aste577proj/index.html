Re-Entry Model

Goal

Create a set of files that model the re-entry of a spacecraft with thrusters and parachutes.

Cause

My EDL (Entry, Descent, and Landing) class in Fall 2023 required us to model the re-entry of a spacecraft for multiple assignments, as well as the final group project. I was able to use the MATLAB files I made for the assignments in the project.

Process

I built off of simplified models that were made for earlier assignments, mainly the parchute re-entry model. The resulting propagator performs the following steps at each time step:

  1. Find the density at the current altitude based on the exponential model.
  2. Determine the accelerations in the x and y directions (from thrust, parachute drag, and spacecraft drag).
  3. Find the new velocity and position in the x and y directions.

Results

The propagator ended working, so I made a set of files that could solve the re-entry propagation for the mass flow rate of the thrusters if the initial and final states are given. To do this, I used a bisection solver I made earlier in the semester.

The code has been uploaded to a GitHub repository.
